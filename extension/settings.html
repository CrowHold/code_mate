When a term isn't in the curated glossary, Code_Mate can ask Claude to generate a definition using your Anthropic API key. Definitions are cached locally so repeat lookups cost nothing. Your key is stored in chrome.storage.local and never leaves your browser. Get an API key →

Use Sonnet for ELI25
Default is Haiku 4.5 for all tiers (~$0.002 per definition). Toggle on to use Sonnet 4.6 for ELI25 (~$0.007). Better answers, slightly slower.
